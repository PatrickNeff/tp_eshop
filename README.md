========== Projet site eshop ==========

Rappel config git:

# Initialisation

cd /le/chemin/de/votre/site
git init
git remote add origin VOTRE_URL
git remote -v
git fetch
git checkout master
git pull

# Utilisation

git add monfichier.txt
git commit -a -m "Mon message"
git pull
git push

Lors de l'ajout d'un fichier :
git add FILE
Si jamais vous savez pas où vous en êtes :
git status
